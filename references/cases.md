# 案例库（范本可引用片段 + 句式模式）

> 来源：12 条已审定范本的提炼（公开出版物、公开信、公开演讲及匿名化的内部范本）。引用时标注出处。

## 判断句三式（原则/价值判断——钉死不留缓冲）

- "X 是 Y"钉死型："懒惰是万恶之源"（张一鸣）
- 比较级排序型："选择比努力更重要，观念比选择更重要""强烈的动机比方法更根本"（张一鸣）
- 对偶双断型："短期交往说话忽悠会有溢价，长期交往说话实在会有溢价"（张一鸣）

## 置信度判断式（预测/不确定判断——先标置信度）

- "我模糊的认为……""大概率""恐怕"（黄峥）——重判断前自我降格，几乎不用"一定/必须"
- 对称短句仍可用："不是从打破一个垄断到创造一个新的垄断，而是从打破一个垄断到提供一个新的选择"（黄峥 2018 股东信）

**分界**：原则与价值判断用钉死式，预测与不确定判断用置信度式。用反的两种病：把预测说成铁律（装全知）、把原则说得含糊（没立场）。

## 概念命名四则

- 具体物命名 + 场景化定义："电影院现象——前排的人站起来了，后排的人也会站起来"（黄峥）
- 比喻锚定 + 机制展开："规模效应就像万有引力，足够大就把其他物质吸过来围绕自己转"（王慧文）
- 等式定义："Agent = Model + Harness——模型负责思考，Harness 负责让思考可理解、可协作、可复现、可长期运行"（周枫）
- 常用字框架 + 命名即判断："点线面体""三浪叠加"（曾鸣）

## 判断标准的给法

- 数字阈值不给形容词："渗透率超过 50% 之后市场格局要变就很难了"（王慧文）
- 判断配掉坑后果："判断错了，在一个不够大的市场里投入太多钱，就会掉到坑里"（王慧文）
- 临界点思维："质变就在这 1% 到 5% 之间"（黄峥）
- 问题自带判断标准："编译快了 50%？还是 P99 降了 20%？必须有量化指标"（某大厂年度技术规划讨论稿）

## 提炼与删改的纪律

- "写完后至少看两遍，竭力将可有可无的字、句、段删去，毫不可惜"（鲁迅，毛引用并加码"重要的文章不妨看它十多遍"）
- "好备忘录的叙事结构会强制你想得更透，强制你真正搞清楚什么比什么更重要"（Bezos 2004 邮件）
- "凡真理都不装样子吓人，它只是老老实实地说下去和做下去"（毛《反对党八股》）
- "变化之妙，要有常态衬托才显得出来。一旦常态不存，余下的只是乱，不是变了"（余光中）

## 抓重点

- "研究任何过程……就要用全力找出它的主要矛盾。捉住了这个主要矛盾，一切问题就迎刃而解了"（《矛盾论》）
- "任何质量都表现为一定的数量，没有数量也就没有质量"（《党委会的工作方法》）
- "射箭要看靶子，弹琴要看听众，写文章做演说倒可以不看读者不看听众吗？"（《反对党八股》）

## 阶段性结论与数据纪律（某大厂内部总结/规划稿，已匿名化、数字已虚构）

- 证明强度分级："初步证明联营模式可以基本盘活尾部网点；但试点仅两个月，尚不能证明该模式可规模化复制"——试点结果只领"初步证明"，并写明还差什么
- 负面同句归因："分享路径缩短 63%（16 步→6 步），但试点使用率低于 10%，核心原因是素材不足加工具使用门槛"——成绩与失败同句，归因紧跟数字
- 拐点归因："6 月渠道调整划走网点，渗透率跌到 8%；11-12 月考核口径回归，大幅回升；1 月疫情仓配关停，跌到 7.6%"——曲线每个拐点有名有姓
- 反思真话："该方向投入较多但收效甚微；考虑后续非核心能力直接内嵌，并接受部分体验下降"——承认打水漂并给止损决策，不写"继续加强"
- 决策留痕："不优化框架本身而用 Java 重写的原因说明：框架优化后升级要背两边代码合并成本；开源框架性能优化投入 ROI 低；重写后可复用的基础设施更丰富"——被否方案连同理由写出

## 开篇范例

- 两短句下定义 + 具象替代抽象："拼多多不是一个传统的公司。它在大家都觉得电商的格局已定，历史书已经写完的时候诞生。"（黄峥招股书信开篇——"历史书已经写完"替代"市场成熟"）

## 收束范例

- 延展比喻收束全篇："在森林里，每一个局部树和树的竞争是激烈的……但整个森林最终所有树的方向又是一致的，那就是向着阳光的方向。"（黄峥 2019 股东信）
- 器官系统统一多点："上下文管理是心脏、工具是手、评测是免疫系统、循环编排是骨架——七点其实是同一个工程的七个侧面。"（周枫）
