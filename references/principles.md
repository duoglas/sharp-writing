# 六原则总纲

每条原则 = 一句可执行的要求 + 违反时的典型病 + 范本出处。写作中和 REVIEW 时都对照。

## 1. 靶子先行

动笔前写一行：给谁看、他已知什么、看完做什么决策。写完用"转发测试"验证：读者的上级读它需要二次翻译吗？

- 病：不想读者就动笔；内部黑话、编号（KR1/T3）直接怼给外层读者
- 源：毛《反对党八股》"射箭要看靶子"；Amazon PRFAQ 从新闻稿倒推；季米特洛夫"要想到你究竟为什么人写东西"

## 2. 主要矛盾一句话

先列全所有事项，强制挑出对目标起决定作用的一件做主线，其余降级为一句话点到。每件事先给三七开定性（成绩为主还是问题为主），定性句放段首。

- 病：平均用力、"如堕烟海找不到中心"；bullet 罗列当思考（"甲乙丙丁开中药铺"——几条之间没有内部联系、没有统领判断）
- 源：《矛盾论》主要矛盾；《党委会的工作方法》"成绩究竟是三分还是七分"；结构思考力"以上统下"

## 3. 结论先行，标题即论点

结论放最前，标题写成判断句而不是主题词——只读标题连起来能得到完整论点链（"Harness即产品"式）。每个上层结论用"疑问-回答"测试：读者会问什么？下层内容是否恰好回答了它？

- 病：标题写"XX 分析""XX 情况"（主题词不是论点）；结论藏在最后
- 源：周枫（七个小节标题全是判断句）；金字塔原理；Duarte 数据页标题直接写结论句

## 4. 数字带基线，优先临界点

每个数字带 vs（vs 目标 / vs 历史 / vs 对照组）；定性词（偏慢/良好/明显）必须有数量支撑，说不出占比和及格线就删定性只留事实。关注临界点而非平均值。

- 病：孤立数字（"120 提交 2.1 万行"——和什么比？）；光秃秃的"进展良好"
- 源：《党委会》"胸中有数——没有数量就没有质量"；黄峥"质变就在这 1% 到 5% 之间"；王慧文判断标准全给数字阈值（"渗透率超过 50% 格局就难变"）

## 5. 概括仍具体

概括的正确产物是"更短但同样具体"。逐个概括句过三查：抽象状态词能否就地反推出事实（不能就退回事实句）；对仗/金句全篇 ≤1 处；定性词带基线。

- 病：把事实压成标签（"机制成型""组织级可观测"）；"到抽象名词里兜一圈回来，门面堂皇内容空洞"（余光中）
- 源：概括三查（2026-07 反馈固化）；鲁迅"竭力将可有可无的字、句、段删去"

## 6. 判断压成一句，按性质选句式

重要判断压进一句可摘引的短句。两种句式按判断性质选：

- **原则/价值判断**用张一鸣式——钉死不留缓冲："X 是 Y"型、比较级排序型（"选择比努力更重要"）、对偶双断型
- **预测/不确定判断**用黄峥式——先标置信度再下判断："大概率 / 恐怕 / 我模糊的认为"，几乎不用"一定/必须"

用错方向的病：把预测说成铁律（装全知），把原则说得含糊（没立场）。

## 附：概念命名规范（造新概念时）

1. 用具体物命名，随附一个场景化定义，禁止裸造抽象名词——"电影院现象：前排的人站起来了，后排的人也会站起来"（黄峥）
2. 比喻锚定后展开机制，在喻体内完成推理闭环——"规模效应就像万有引力，足够大就把其他物质吸过来"（王慧文）；"模型是可更换的引擎，Harness 才是自己造的车"（周枫）
3. 框架词用最常用汉字，带空间/身体隐喻，命名即判断——"点线面体""三浪叠加"（曾鸣）
4. 比喻可以做段落骨架，但整段推理必须在喻体内自洽（黄峥"储蓄罐里的钱不该存定期"）
